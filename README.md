> **A fully functional, single-file AI-powered news application that curates, summarizes, and analyzes global news in real-time.**

![NeuralNews AI Screenshot](screenshot.png)

## ✨ Features

- **🤖 AI-Powered Summarization** — Extractive NLP algorithm generates concise summaries with key point extraction
- **📰 Curated News Feed** — Integrates with Actually Relevant API for pre-curated, significance-ranked global stories
- **🎭 Sentiment Analysis** — Real-time emotional tone detection (Positive/Negative/Neutral)
- **🔖 Smart Bookmarks** — Persistent local storage for saved articles with quick-access drawer
- **🔊 Text-to-Speech** — Built-in Web Speech API integration for hands-free listening
- **📤 Multi-Format Export** — Export articles as JSON, CSV, or plain text
- **🌓 Dark/Light Mode** — Fully adaptive theme system with glassmorphism design
- **⌨️ Keyboard Shortcuts** — `Ctrl+K` search, `Ctrl+R` refresh, `Esc` close modals
- **📱 Fully Responsive** — Optimized for desktop, tablet, and mobile devices
- **⚡ Offline Support** — Service Worker for basic offline functionality
- **🎯 Category Filtering** — Science & Tech, Human Development, Planet & Climate, Existential Threats
- **🔍 Real-time Search** — Instant article filtering by title, content, or source

## 📦 Installation

No build step required! Simply open `index.html` in any modern browser.


# Clone the repository
git clone https://github.com/your-username/neuralnews-ai.git

# Navigate to directory
cd neuralnews-ai

# Open in browser (or use Live Server)
open index.html
🎮 Usage
Browse News — Articles load automatically from curated sources
Filter by Category — Click category chips to narrow by topic
Search — Use the search bar or press Ctrl+K to find specific topics
AI Summary — Click "AI Summary" on any card to generate intelligent analysis
Save Articles — Click the bookmark icon to save for later
Listen — In the summary modal, click "Read Aloud" for TTS
Export — Use the download menu to export your feed as JSON/CSV/TXT

neuralnews-ai/
├── index.html          # Single-file application (HTML + CSS + JS)

State Management: Centralized state object with reactive UI updates
API Layer: Fetch wrapper with fallback to demo dataset
Summarization Engine: Client-side extractive algorithm (no external API keys needed)
Storage: localStorage for bookmarks, theme, and usage statistics
🔑 Key Implementation Details
Extractive Summarization — TF-IDF inspired scoring with position boosting for lead sentences
Sentiment Analysis — Lexicon-based scoring with positive/negative keyword matching
Key Point Extraction — Pattern matching for significance markers (breakthrough, discovered, etc.)
Responsive Grid — CSS Grid with auto-fill and featured article spanning
